发表于 2017-05-30 | |

##环境配置
需要安装的工具包括：Node.js, Git, Hexo。

####1. 安装Node.js
用来生成静态页面，建议Node.js官网下载，然后一路安装即可。

####2.安装git
用来将本地Hexo内容提交到Github上。Xcode自带Git，这里不再赘述。或者点击安装程序来下载安装git.
